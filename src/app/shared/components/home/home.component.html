Frequently Asked Questions

What is Mamba UI?

Mamba UI is a free, open-source collection of beautiful UI elements based on Tailwind CSS that make it easy for anyone to build a website in minutes. We're building Mamba as a place where people can easily access the UI elements they need for their next project and to share what they've created with the design community.

What does this cost?

Nothing! All of this is MIT licensed so you can use them in your own projects as you wish. If you'd still like to support me you can star this project on Github. Or if you are feeling really generous you can also buy me a coffee.

We also have a couple of new and exiting premium features planned. You can find more information about the upcoming features and their release schedules in our  product roadmap.

When are you releasing new stuff?

The project is in active development and we are creating more elements and templates as fast as we can. If you have a specific component, section or template that you need, just message us via Github or Twitter and we'll see how we can help.

If I use these in my projects do I have to give credit?

No, crediting is not mandatory and you don't have to mention that you used Mamba UI. But if you find our components and templates helpful you can give us a shoutout.

Some of the colors don't seem to work. Is something broken?

Tailwind doesn't have all of their available colors in the default configuration. You can check out our sample tailwind configuration in the getting started -section. That has all the different colors enabled so you can copy-paste the config from there to get everything to work as they should.
