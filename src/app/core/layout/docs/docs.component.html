Create a beautiful website in minutes

Get a jumpstart on creating your new webpage with our fully responsive and carefully styled components and sections. You can get the layout of your website done with just a couple of clicks.

All the components are just good ol' HTML and CSS, but without the need of writing complex css rules thanks to TailwindCSS. TailwindCSS is a utility-first CSS framework so if one of our components doesn't fit your needs just right, you can modify them easily by following the TailwindCSS documentation.

Features

  • Idividual components that can be re-used multiple times in your designs.
  • Pre-made building blocks that you can stack on top of each other like Legos to build a website of your own in minutes.
  • Full templates that showcase pieces of what you can achieve with the building blocks that are in this UI Kit.

How to use the components on your own websites?

  1. Find something that you like from the components, sections or full pages.
  2. Choose the primary color from the available colors in the footer
  3. Select the light or dark theme
  4. Click the "HTML"-button to see the code for the selected part
  5. Copy/paste into your project
  6. Profit!

Prerequisites

Since everything you see on this page is built with TailwindCSS you'll need to have it installed for the components and sections to work out-of-the-box on your website. They have an extensive installation guide in their docs for all sorts of different tools and frameworks. If you want to quickly test the components or customize the code, you can also copy them to TailwindCSS's official online playground Tailwind Play.

Also if you want to use the exact same icons as I have, you'll need to install CoreUI Icons as well. And most of the text on the site uses the Montserrat font-family.
