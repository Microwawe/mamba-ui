Create stylish websites by copy-pasting components

All the components are just good ol' HTML and CSS, but without the need of writing complex css rules thanks to TailwindCSS. TailwindCSS is a utility-first CSS framework so if one of my components doesn't fit your needs just right, you can modify them easily by following the TailwindCSS documentation.

Features

There are dozens of individual components and page sections that you can just copy-paste you your own website.

Prerequisites

Since everything you see on this page is built with TailwindCSS you'll need to have it installed as well for the components and sections to work out-of-the-box. They have an extensive installation guide in their docs for all sorts of different tools and frameworks.

Also if you want to use the exact same icons as I have, you'll need to install CoreUI Icons as well. And most of the text on the site uses the Montserrat font-family.
