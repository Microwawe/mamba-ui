Getting started

How to use the components on your own websites?

  1. Find something that you like from the components, sections or full pages.
  2. Choose the primary color from the Tailwind CSS color palette
  3. Select the light or dark theme
  4. Click the HTML, Vue or JSX -buttons to see the code for the selected part
  5. Copy/paste into your project
  6. Profit!

Prerequisites

Since everything you see on this page is built with Tailwind CSS you'll need to have it installed for the components and sections to work out-of-the-box on your website. They have an extensive installation guide in their docs for all sorts of different tools and frameworks. If you want to quickly test the components or customize the code, you can also copy them to Tailwind CSS's official online playground Tailwind Play.

Tailwind CSS has 17 different colors in total but by default only 8 of them are available in the configuration. We have inlcluded all the available colors in the Tailwind CSS color palette so you can try all of them out. If you use one of the colors that isn't in the default configuration you will need to add it to your tailwind.config.js file.
